# Disney+ Clone

## Description

This project is a **Full Stack Disney+ Clone** built using **React.js, Tailwind CSS, Vite, Node.js, and Express.js**.
It was created by following a tutorial to learn how to build a modern full-stack web application.

The project includes a **React frontend for the UI** and a **Node.js + Express backend API**. The frontend communicates with the backend to fetch and display movie data, similar to how streaming platforms work.

This project is aimed at beginners who want to understand the basics of **full-stack development with React**.

---

## Features

* Disney+ inspired user interface
* Built using **React.js with Vite**
* Styled with **Tailwind CSS**
* Backend API using **Node.js and Express.js**
* Frontend connected to backend API
* Component-based React structure
* Beginner-friendly full stack architecture

## Getting Started

### 1. Clone the repository

```
git clone https://github.com/yourusername/disney-clone.git
```

### 2. Install dependencies

Frontend:

```
cd client
npm install
```

Backend:

```
cd server
npm install
```

### 3. Run the project

Start the backend server:

```
npm run server
```

Start the frontend development server:

```
npm run dev
```

The application should now be running locally.
